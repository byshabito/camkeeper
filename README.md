## Installation (Development)

1. Clone this repo
   ```sh
   git clone https://github.com/byshabito/camkeeper.git
   ```
2. Open `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the cloned project folder
